## Section 3: Utility Basics

**Holder rights / access:**
Each FRG token represents one prepaid AI agent task run (summarize, classify, transform). The token is consumed (burned) on task execution.

**Product utility:**
1 FRG = 1 AI agent task. Burn-on-use. No passive yield.

**Ecosystem utility:**
FRG is the native spend unit for FORGE AI agent tasks built on HACD Stack.

**Utility at launch:** No — AI agent infrastructure is in development.

**Roadmap dependencies:**
- AI agent task infrastructure deployment
- Token burn mechanism on-chain integration
- Developer API release

**What should not be promised:**
- Price floor
- Investment return
- Exchange listing guarantee
- Fixed utility delivery date
- Staking rewards or yield
